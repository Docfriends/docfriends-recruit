# Question

숫자와 문자가 섞인 q라는 문자열이 있는데

해당 문자열에서 숫자의 배열을 numbers라고 할때

가장 작은 숫자가 numbers의 a번째, 가장 큰 숫자가 numbers의 b번째에 있다고 한다.

a + b의 합을 구하여라.

# Input

  String q = "ab23c4d56e78f9g12h34i5j12k45l67n89m99o1k123p456q567r768s890t67u456v345w234x23y239z";

# Output

  31
